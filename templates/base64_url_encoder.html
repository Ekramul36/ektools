How to Use This Tool

Step-by-step guide +

1. Choose the Base64 or URL tab depending on what you want to convert.
2. Select Encode or Decode mode.
3. Type or paste your text into the Input box — the Output updates automatically.
4. Click Copy Output to copy the result to your clipboard.
5. Use Swap Result to Input to instantly reverse the conversion (e.g. check that decoding gives back your original text).

Frequently Asked Questions

What is Base64 encoding used for? +

Base64 encoding converts binary data or text into ASCII characters. It's commonly used to embed images directly in HTML/CSS, send data in JSON/XML APIs, or safely transmit data over text-based protocols like email.

Why do I need to URL encode a string?

URL encoding (also called percent-encoding) converts special characters like spaces, &, and ? into a safe format for URLs, preventing broken links and query string errors.

Is my data safe using this tool?

Yes — all encoding and decoding happens locally in your browser using JavaScript. Nothing is ever uploaded to our servers.

What happens if I try to decode invalid Base64?

The tool will show an error message letting you know the input isn't valid Base64, so you can double-check your text before converting.
