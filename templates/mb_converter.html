Good to know

Binary vs. decimal storage units

There are actually two competing conventions for what "1 GB" means, and mixing them up is the single most common source of confusion in storage math. Operating systems like Windows and macOS use the binary standard, where each unit is 1024 times the one before it — this converter follows that standard, since it matches what you'll see in a file manager or disk properties window.

Storage device manufacturers, on the other hand, typically label capacity using the decimal standard, where 1 GB is exactly 1,000,000,000 bytes and each unit is 1000 times the one before it. This is why a drive labeled "1TB" often shows up as roughly 931 GB when you plug it in and check its properties — the drive genuinely holds 1 trillion bytes, but your operating system is reporting that same byte count using 1024-based units.

Neither convention is "wrong" — they're just answering slightly different questions. If you're comparing a file size shown by your computer against another file size shown by your computer, the binary standard used here will match exactly. If you're comparing against a number printed on packaging or a marketing spec sheet, keep in mind it's likely using the decimal definition instead, and expect a small gap between the two.

FAQ

Frequently asked questions

How many MB is 1 GB?
1 GB equals 1024 MB. This converter uses binary (1024-based) conversion, which is the standard used by operating systems and storage devices.
How many GB is 1 TB?
1 TB equals 1024 GB using the same binary conversion standard applied throughout this tool.
Is this tool free?
Yes, 100% free. No signup required, and there's no limit on how many conversions you can run.
Why do file managers sometimes show a different GB value than storage labels?
Storage manufacturers typically label capacity using decimal (1000-based) units, while operating systems display it using binary (1024-based) units — this converter uses the binary standard, matching what your computer's file manager shows.
Can I convert Bytes directly to TB?
Yes, select Bytes as the starting unit and the result grid will show the equivalent value in KB, MB, GB, and TB all at once.
Does it handle decimal values?
Yes, you can enter decimal values like 2.5 and the converter will calculate the equivalent across all units accurately.
Is my data uploaded when I use this converter?
No. This is a unit calculator, not a file converter — you only enter a number, and the calculation runs entirely in your browser.
What's the difference between MB and Mb?
MB (capital B) means megabytes, used for file and storage size; Mb (lowercase b) means megabits, commonly used for internet speed — this converter works in bytes, not bits.
How do I convert internet speed (Mbps) using this tool?
This converter works in bytes (B, KB, MB, GB, TB), not bits, so for Mbps to MB/s you'd first divide the Mbps value by 8 to get megabytes per second, then use this tool for any further unit conversion.
Why does the Bytes result show such long decimal numbers?
Smaller units like KB and MB carry many decimal places when converted from a large GB or TB value, since each step divides by 1024 — the extra precision is shown so the exact figure is available if needed.
Can I use this to check if a file will fit on a USB drive or SD card?
Yes, convert the file size and the drive's labeled capacity to the same unit to compare them directly, keeping in mind manufacturers usually label capacity in decimal GB.
Does this tool round the results?
Smaller units are shown with more decimal places and larger units with fewer, so each result stays accurate to a meaningful number of digits for its scale.
