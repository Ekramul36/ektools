❓ Frequently Asked Questions

Can I start numbering from a number other than 1?
Yes, set any starting number in the "Start Number" field — useful for documents that continue from another file.
Will this affect my PDF's existing content?
No, page numbers are added as an overlay without modifying your original content.
Is there a file size limit?
Files up to 50 MB are supported for free.
Is my file kept private?
Yes, your file is processed and then automatically deleted from our server.
