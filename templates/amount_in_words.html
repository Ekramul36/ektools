How to Use This Tool

Step-by-step guide +

1. Type or paste any number into the input box (e.g. 125430.50).
2. Choose Indian (Lakh/Crore) or International (Million) format using the toggle.
3. The amount in words appears instantly below, including paise/decimals if present.
4. Click Copy Result to copy the text — paste it directly onto a cheque, invoice, or document.
5. Try the quick example chips above the result box to see common amounts converted.

Frequently Asked Questions

How do I convert a number to words? +

Type or paste the number into the input box above. The tool instantly converts it into words in both Indian (lakh/crore) and International (million/billion) format, updating as you type.

Can I use this for cheque writing? +

Yes — this tool is built exactly for that. Enter the amount, select Indian format, and copy the result directly onto your cheque. Paise are automatically included when present.

Is my data safe?

Completely. All conversion happens locally in your browser using JavaScript — no numbers are ever sent to our servers.

What's the difference between Indian and International format?

Indian format uses Lakh and Crore (1,00,000 = One Lakh), commonly used in India, Bangladesh, Pakistan and Nepal. International format uses Thousand, Million, Billion — used in most other countries.
