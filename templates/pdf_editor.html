There's no install, no plugin, and no account wall between you and your file. Step-by-step guide

How to edit a PDF online for free

No installs, no watermark, no account. Everything above runs right in your browser — here's exactly what each step does and why it works that way.

Upload your PDF

Drag a file into the upload box above, or click it to browse your computer. The moment the file loads, EkZapp renders every page directly in your browser using the same rendering engine that powers most PDF viewers — so what you see matches what will print. Your file is processed securely and never shared with anyone; nothing is stored longer than needed to complete your edit, and the working copy is deleted automatically once you're done.

Pick a tool from the toolbar

Use Text to click anywhere on the page and start typing — a text box appears exactly where you clicked, ready to type into immediately. Use Image to drop in a logo, photo, or scanned stamp; it lands near the middle of the visible page and you drag it into position from there. Every element you place, whether text or image, can be dragged to reposition and resized once it's on the page, so getting the exact spot right doesn't require starting over.

Format as you go

With the Text tool active, the formatting bar above the page lets you change font family, size, and color for whatever's currently selected. You can set these before you start typing a new box, or select an existing one afterward and adjust it — both work the same way, so there's no wrong order to do things in.

Apply changes and download

Once your edits look right, click Apply changes. EkZapp flattens every addition — text, images, and anything else you placed — directly into the PDF's page content and hands you a clean file to download. There's no watermark stamped across it and no login wall between you and the finished document; it's simply your original file with the new content baked in.

A few tips

Getting a cleaner result

Zoom in before placing small text

Use the Zoom + button in the toolbar to get a closer view of the exact spot you're editing — it makes lining text up with existing content on the page noticeably easier than working at the default size.

Match the existing font where it matters

For anything formal — contracts, certificates, official forms — picking a font close to what's already on the page keeps your addition from standing out as an obvious edit.

Use a transparent PNG for logos

A logo exported with a transparent background sits naturally on the page; a logo with a white background box will visibly cover whatever's underneath it.

Double-check before applying

Once you click Apply changes, the edits are flattened permanently into the file. Take a moment to review placement and spelling on every page first — it's the one step worth not rushing.

Good to know

What this version doesn't do yet

Being upfront about scope saves you a wasted upload. With the Text tool active, clicking on text that's already in the PDF lets you edit it in place — the original is erased and your new text takes its exact spot, matched to the detected size and position. This only works on text-based PDFs; a scanned or image-only page has no underlying text to detect, so on those pages you can still add new text on top, just not edit what's baked into the scan.

Font matching is a best-effort guess, not a perfect read of the original file's embedded font — the size and position lock on accurately, but you may want to double check the font family looks right before applying, especially on documents using an unusual typeface.

If your document is a scanned image rather than a text-based PDF, adding new text on top still works fine — you're placing a new layer on the page, not modifying the scan itself. What won't work is selecting and changing text that's part of the scanned image, since there's no underlying text to select in the first place.

Frequently asked questions

PDF Editor FAQ

Is the EkZapp PDF editor really free?

Yes. Uploading, editing, and downloading your PDF is free, with no watermark added to your file. Free-tier limits apply to file size and how many tasks you can run per hour.

Can I edit text that already exists in the PDF?

Yes — with the Text tool selected, click directly on any existing text and it becomes editable in place. The original is erased and your edit is drawn in the same spot at a matching size. This only works on text-based PDFs; scanned pages have no underlying text to detect.

Will my file be shared with anyone?

No. Your PDF is only used to generate your edited download and is not shared with third parties or used for any other purpose.

Does this work on mobile?

Yes, the editor is fully responsive and works in any modern mobile browser, though for detailed editing a larger screen is easier to work with.

What file types can I upload?

Currently only PDF files. If you need to add an image to a PDF, use the Image tool inside the editor rather than uploading the image directly.

Is there a limit on how many pages I can edit?

You can add text and images across every page of your document within the free tier's file size limit — there's no separate page cap for editing.

Can I undo a change after adding it?

Yes, use the Undo button in the top bar to remove the most recent element you placed, or click the delete icon that appears on a selected text box or image to remove it directly.

Do I lose quality when I download the edited PDF?

No. Applying changes only adds your new text and images to the page — it doesn't recompress or re-render the rest of the document, so the original content stays at its original quality.

Can I edit a password-protected PDF?

Not directly — you'll need to remove the password first using a PDF unlock tool, then upload the unlocked file here. This keeps the editor from ever needing to store or process your document's password.

Why does my text look different after I click Apply changes?

The formatting bar preview and the flattened output use the same font rendering, so text should look consistent. If a font you selected isn't available in the final render, the editor falls back to the closest built-in match — sticking to the default font options avoids this entirely.
