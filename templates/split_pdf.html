One field, two ways to trim a PDF

Splitting a PDF really comes down to one decision: do you need a single page, or a continuous block of pages? Typing a plain number like "5" pulls out exactly that one page into its own file — useful for a signature page, a single invoice, or one form buried inside a longer packet. Typing a range like "12-18" instead pulls out every page from the first number to the second, inclusive, which is what you'd use for a chapter, a section, or a set of related pages that belong together. Either way, the extraction happens against a fresh copy of the file; the original PDF you uploaded is read but never altered, so if you need a different page or range later, you can always come back and run it again from the same source. Knowing which of the two you actually need before typing anything in saves a wasted attempt — a quick glance at the PDF in any viewer to confirm the exact page numbers is usually enough to get it right on the first try.

Why splitting a PDF matters

 

PDFs tend to accumulate everything related to a subject into one file, even when only a fraction of that file is relevant to whoever's receiving it. A 40-page annual report might only need its financial summary shared with a colleague; a 200-page scanned court filing might only need its signed affidavit sent to a clerk. Splitting turns "send the whole thing and let them find it" into "send exactly the page they asked for" — smaller files, less scrolling, and no risk of exposing content that has nothing to do with the request. If the goal is actually the opposite — trimming a few unwanted pages out of an otherwise-complete document rather than extracting a section — Delete Pages is usually the better fit.

This matters more than it might seem for anything containing personal or financial details. A bank statement split down to just its summary page shares far less incidental information than the full multi-page original — account activity from unrelated months, other account numbers on the same statement, or personal details that simply weren't asked for. The habit of sending only what's needed, rather than the whole document by default, is a small but genuinely useful privacy practice.

It's also just faster for whoever's on the receiving end. A one-page extract opens instantly and is unambiguous about what it contains; a 40-page bundle with a note saying "see page 12" asks the reader to do the navigating themselves. For anything that gets forwarded, filed, or referenced later, a properly split file is easier to find again too — it's self-contained, correctly named, and doesn't require remembering which page of a much larger document mattered. And once you've split out the pages you need, tools like Merge PDF and Compress PDF come in handy for putting a new set of extracted pages back together or shrinking the result before sending it on.

Tips for splitting a PDF cleanly

 

Before typing a page number, open the PDF once in any viewer and confirm the exact page you need — some viewers number a cover page as "page 0" or skip counting a blank page, so what you see on-screen doesn't always match what a straightforward page count would suggest. A ten-second check here avoids a wasted attempt.

If you only need non-consecutive pages — say, page 2 and page 9 from the same document — this tool extracts one continuous range at a time, so the simplest approach is to run it twice and keep both results, or use the Delete Pages tool instead and remove everything except the pages you want to keep in one pass.

For documents you split regularly — a monthly statement, a recurring form — it helps to note down which page number you typically need once you've found it. Bank statements and utility bills, for example, are usually laid out consistently from month to month, so the summary page tends to sit in the same spot every time, saving you from recounting it on the next round.

Common problems when splitting a PDF

 

"Page number doesn't exist" errors. This usually means the page count assumption was off by one — recount directly in a PDF viewer rather than relying on memory, especially for documents with a cover page or blank pages that are easy to miscount.

Entering the wrong range format. A list of individual pages like "2, 5, 9" isn't the same as a continuous range like "2-5" — this tool expects a single page number or one continuous block of pages, not a comma-separated list.

Trying to split a password-protected PDF. A locked file needs its password removed first with a PDF unlock tool before any page-extraction tool, including this one, can open and read it.

Assuming the split file keeps everything from the original. Bookmarks, comments, or form fields tied to pages outside your selected range won't carry over — the new file only contains what's on the pages you extracted.

Frequently asked questions

 

How do I extract only one page from a PDF?

Just enter the page number in the field. For example, type "5" to extract only page 5 from your PDF.

Can I extract multiple pages at once?

Yes. Enter a page range like "3-7" to extract pages 3 through 7 into a single new PDF.

Will my original PDF be changed?

No. Your original PDF is never modified. A brand new PDF is created with only the pages you selected.

Is this tool free?

Why would I need to split a PDF instead of sharing the whole file?

Splitting lets you share only the relevant pages, such as one chapter, one invoice, or one form, without sending the entire document.

Do I need to install any software?

No installation is needed. The tool works entirely in your browser on any device, including mobile phones.

What happens if I enter a page number that doesn't exist?

The tool shows an error asking you to check the page number against your PDF's actual page count, rather than producing a broken or empty file.

What's the difference between Split PDF and Delete Pages?

Split PDF keeps only the pages you specify and creates a new file with just those. Delete Pages does the opposite — you remove the pages you don't want, and it keeps everything else.

What is the maximum file size?

PDF files up to 50MB are supported.

Can I split password-protected PDFs?

Not directly — a locked PDF needs its password removed first with a PDF unlock tool. Once unlocked, the resulting file can be split normally.

Can I split scanned PDFs?

Yes. Splitting works on the page level regardless of whether a page contains real text or a scanned image, so scanned PDFs split just as reliably as text-based ones.

Can I extract odd or even pages only?

Not directly with a single entry — this tool extracts one continuous range or a single page at a time. For alternating pages, the more practical route today is running the tool once per page you need.
